# Getting Started with Node App as server for QUBE assignment

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the server in the development mode.\
Open postman [http://localhost:8000](http://localhost:8000) followed by paths.


[http://localhost:8000//api/v1/appliances](http://localhost:8000//api/v1/appliances) 
to get all appliances

[http://localhost:8000//api/v1/appliances/:serialNo/info](http://localhost:8000//api/v1/appliances/:serialNo/info)
to get specific appliance information

[http://localhost:8000//api/v1/add_appliances](http://localhost:8000//api/v1/add_appliances)
to add 100 entries with random datas for each hit of this endpoint
